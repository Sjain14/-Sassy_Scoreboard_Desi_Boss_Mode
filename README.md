## 🛑 The Problem
Modern sports scoreboards are clinically boring. They show you numbers, strike rates, and graphs, but they completely ignore the most critical metric in franchise cricket: **The ROI (Return on Investment).** Nobody tells you that the player making ₹17 Crores just played a dot ball and is essentially stealing company time. 

## 🛠 The Solution
Enter the **Sassy Scoreboard**. We ditched the traditional sports UI and rebuilt it as a high-octane financial dashboard. By piping live cricket data into Google's Gemini 1.5 Pro, we generate brutal, real-time corporate performance reviews for every single delivery. It’s a ruthless appraisal cycle, delivered ball-by-ball.

---

## ✨ Core Features

### 🏢 "The Boss's Cabin" (Real-Time AI Performance Reviews)
Every delivery triggers a real-time AI evaluation using Google Gemini 1.5 Pro. The system analyzes the player's auction price, current strike rate, and ball outcome to generate a strictly unique, under-150-character corporate roast. Get ready for PIPs (Performance Improvement Plans) and instant terminations.

### 📉 ROI Meter
A dynamic financial gauge measuring player value against their exorbitant auction price. Watch the meter fluctuate between "Acceptable Output" and "Notice Period Imminent" in real time as the innings progresses.

### 🎭 "12-Ball Climax" (Narrative Demo Mode)
API rate limits during your hackathon demo? Not an issue. Trigger the **12-Ball Climax Mode**—a pre-cached, offline simulation of the final two overs of the legendary GT vs DC (April 8, 2026) matchup where DC needs 36 off 12. Management is watching.

### 📱 Pulse-𝕏 Publishing
A simulated automated social broadcasting engine. Every time a new AI roast is generated, the UI pulses with a "Live on 𝕏" badge, creating the ultimate illusion of an automated, trash-talking corporate Twitter account.

## 💻 Tech Stack & Methodology

We utilized the **F.L.O.W. Framework** (Frame, Layout, Orchestrate, World) alongside the **Stitch MCP** for rapid, agent-driven UI synthesis.

| Layer | Technology | Purpose |
| ------------- |:-------------:|:-------------|
| **Core** | Next.js 14 (App Router) | Server-side rendering, API routing, and hydration. |
| **Logic** | Zustand | Managing the complex, tick-based offline match state machine. |
| **Styling** | Tailwind CSS & CSS Modules | Midnight Stadium aesthetics, glowing gradients, and glassmorphism. |
| **Animation** | CSS Keyframes & Framer Motion | Micro-interactions, Pulse-𝕏 badges, and fluid layout shifts. |
| **Brain** | Google Gemini 1.5 Pro API | Generating hyper-contextual, sarcastic corporate evaluations. |

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sjain14/-Sassy_Scoreboard_Desi_Boss_Mode.git
   cd -Sassy_Scoreboard_Desi_Boss_Mode
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file in the root directory. You will need a Google Gemini API key to run the live evaluation mode.
   ```env
   GEMINI_API_KEY="your-gemini-api-key-here"
   ```

4. **Launch the Development Server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the scoreboard. Don't be late. Management hates tardiness.
